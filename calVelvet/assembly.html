Description

This track shows the sequences used in the July 2015 Calotropis procera genome assembly.

Genome assembly procedures are covered in the NCBI assembly documentation.
NCBI also provides specific information about this assembly.

The definition of this assembly is from this AGP file. The NCBI document AGP Specification describes the format of the AGP file.

In dense mode, this track depicts the contigs that make up the currently viewed chromosome.

Component types found in this track (with counts of that type in parenthesis):

  • D - 'draft' sequence (20530)
